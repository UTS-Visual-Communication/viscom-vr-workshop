For This Task

This brief aims to be a basic introduction to designing for virtual environments and a how we can use a “code library” - A-Frame - to use the basic web skills we already have to create a simple virtual reality experience.

You will design a simple virtual reality experience that lives on the web, based on the idea of a traditional model of the solar system. This is only a place to start and you can play and run with it as you wish.

The following outlines the process, although your studio leaders will also run you through the process in class.

Groups

Get into groups of 2 to 3, try and form groups with at least one person who felt confident with the web (HTML) components of the course earlier this semester.

Glitch

We will be using the Glitch platform to develop your project. In order to access your project and continue working on it after class you need to make an account, which you can do by either logging in with Github or facebook. You can choose either of these options, though making a github account now is a good idea if you think you will be working with code any time in the future. The following instructions assume you will be taking the github path, though feel free to log in with facebook if you so choose. I don’t recommend tying online services to your facebook account, this is up to you and it will work the same as using github.

  • Make an account at http://github.com
  • If you are interested check out the ‘education pack’ offered by github (though this is not needed for our workshop) https://education.github.com/pack
  • Go to http://glitch.com and login using your github credentials.
