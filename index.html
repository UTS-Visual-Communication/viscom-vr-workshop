A-Frame VR

http://aframe.io

We will be using AFrame, a javascript library for creating web based VR experiences. You will have heard me speak about this in the lecture, but your studio leaders will also run through some of the basics with you, more than enough to get you up and running to create your first VR experience.

This is a very powerful library that offers a lot of flexibility and allows you to write complex interactions using javascript, though today we will be looking at working exclusively with HTML elements. (Next year in Narrative, Form and Time you will learn to write interactive code using P5.js - the javascript library you looked at in first semester)

Adding your own assets to the glitch project

To add images you will use as textures via glitch, you need to use the Glitch assets system

  • Within the Glitch editor for your project, click the "assets" button in the left hand column.
  • When this opens you will see some assets we have provided for you to experiment with. To use one of these click the image, then the "copy URL" button and use the URL directly ion your code. (ie: in the "A-Assets" section of your index.html code)
  • To add your own image (.jpg or .png) click the "Add Asset" dropdown button in the top left of the panel ands follow the imntructions. Then use this in the same way by getting its URL using the "copy URL" button.
  • Remebner to check out the comments in the example "index.html" file for more info on the way you use images in A-Frame
