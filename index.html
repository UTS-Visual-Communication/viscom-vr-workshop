For This Task

This brief aims to be a basic introduction to designing for virtual environments and a how we can use a “code library” - A-Frame - to use the basic web skills we already have to create a simple virtual reality experience.

You will design a simple virtual reality experience that lives on the web, based on the idea of a traditional model of the solar system. This is only a place to start and you can play and run with it as you wish.

The following outlines the process, although your studio leaders will also run you through the process in class.

Requirements / How to do This Task

We will be giving you a base file to work on that includes the 'Rhythm' essay. Stick to the base code, and using what we teach you in class we want you design the details of the page

  • Use CSS (Cascading Style Sheets) to alter the typography of the layout. You should seek to create a visual hierarchy in the typography (i.e. using the typographic detailing and white space)
  • Design two images that illustrate or visualise two ideas in the text. Include these images in your web page.
  • You can use the generic web fonts (i.e. ariel, times, verdana). But you should also aim to use web fonts. As they are free and accessible we will be using Google Web Fonts in class and recommend you stick to them for your assignment (for now!)

Working on Your Design

You can work directly with the code in order to achieve your design. Sometimes however it can be quicker to sketch out on paper what you are imagining it will look like. You can use thumbnail sketches to quickly detail the layout and visual hierarchy of your page.
